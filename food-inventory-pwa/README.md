# 食材庫存小管家

可部署的 Next.js 手機版 PWA 專案，包含：

- 食材庫存管理
- 保存期限顯示
- 食譜推薦
- localStorage 本地儲存
- manifest + service worker
- 可加入手機主畫面，像 App 一樣點開

## 本機執行

```bash
npm install
npm run dev
```

打開 `http://localhost:3000`

## 部署到 Vercel

1. 把整個資料夾上傳到 GitHub
2. 到 Vercel 匯入專案
3. Framework Preset 選 Next.js
4. Deploy

## 手機安裝

### iPhone
Safari 開啟網址 → 分享 → 加入主畫面

### Android
Chrome 開啟網址 → 選單 → 加到主畫面
